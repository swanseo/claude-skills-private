# VNCCS_PoseStudio: 의자에 앉아 턱을 괴는 포즈 설정 가이드

## 포즈 개요

의자에 앉아 턱을 괴는 포즈는 다음 요소들로 구성됩니다:
- 상체가 약간 앞으로 기울어진 착석 자세
- 한쪽 팔꿈치를 테이블/무릎에 올리고 손으로 턱을 받침
- 반대쪽 팔은 자연스럽게 내리거나 다리 위에 놓음
- 머리가 약간 앞으로 숙여지며 한쪽으로 기울어짐

## 조정 순서 (권장)

1. **pelvis** 먼저 위치 설정 (앉은 자세의 기준점)
2. **thigh → calf → foot** 순서로 다리 자세 설정
3. **spine_01 → spine_02 → spine_03** 순서로 척추 기울기 설정
4. **neck_01 → head** 순서로 머리 위치 설정
5. **clavicle → upperarm → lowerarm → hand** 순서로 팔 설정 (오른팔 먼저)

## ControlNet 강도 설정

### 권장 강도: **0.65 ~ 0.80**

| 강도 | 효과 | 사용 상황 |
|------|------|-----------|
| 0.5 이하 | 포즈 반영 약함, 캐릭터 자유도 높음 | 포즈보다 캐릭터 외형이 더 중요할 때 |
| 0.65 | **자연스러운 균형** | 일반적인 경우 권장 시작값 |
| 0.75 | 포즈 충실도 높음 | 복잡한 팔 위치를 정확히 재현할 때 |
| 0.80 | 강한 포즈 강제 | 포즈가 반드시 일치해야 할 때 |
| 0.9 이상 | 포즈 과도하게 강제, 어색해질 수 있음 | 특수한 경우에만 사용 |

### 추가 팁

- **Start percent: 0.0, End percent: 1.0** (전체 스텝에 적용)
- 얼굴 디테일이 중요하면 **0.65**부터 테스트
- 손 포즈가 잘 안 나오면 **0.75~0.80**으로 올리기
- 옷/머리카락이 부자연스러우면 강도를 **0.05씩 낮추기**

## 자주 발생하는 문제와 해결

| 문제 | 원인 | 해결 방법 |
|------|------|-----------|
| 손이 얼굴에 묻힘 | hand_r Z값 과도 | hand_r Z를 10~15 정도로 줄임 |
| 팔꿈치 방향 이상 | upperarm_r Y값 조정 필요 | Y를 -10 ~ -30 범위에서 테스트 |
| 상체가 너무 숙여짐 | spine_02/03 X값 과도 | X를 5씩 줄여가며 조정 |
| 머리가 너무 기울어짐 | head Z값 과도 | head Z를 3~5 정도로 줄임 |
| 다리 자세 부자연스러움 | thigh X가 부족 | thigh X를 75~85 범위에서 조정 |

## 참고: 왼손으로 턱을 괴는 경우

오른팔/왼팔 설정을 반전하면 됩니다:
- 위의 `_r` (오른쪽) 설정을 `_l` (왼쪽)에 적용
- 위의 `_l` (왼쪽) 설정을 `_r` (오른쪽)에 적용
- spine_03의 Z값 부호를 반전: `-3` → `+3`
- head의 Z값 부호를 반전: `+8` → `-8`
