# VNCCS_PoseStudio: 의자에 앉아 턱 괴는 포즈 가이드

## 포즈 설명

의자에 앉아 한 손으로 턱을 괴는 자세 (생각에 잠긴/사색하는 포즈).
상체는 약간 앞으로 기울고, 팔꿈치는 무릎 또는 의자 팔걸이에 올려진 형태.

---

## Step 1: 본(Bone) 조정 값

### 기본 앉기 자세 (하체)

```json
"pelvis":    [15, 0, 0],
"thigh_l":   [85, 5, 0],
"thigh_r":   [85, -5, 0],
"calf_l":    [-85, 0, 0],
"calf_r":    [-85, 0, 0],
"foot_l":    [20, 0, 0],
"foot_r":    [20, 0, 0]
```

- **pelvis**: X+15 → 앉을 때 골반이 약간 앞으로 기울어지는 효과
- **thigh_l/r**: X+85 → 허벅지를 수평(90도)에 가깝게 접어 앉은 자세 구현
- **calf_l/r**: X-85 → 종아리를 아래로 내려 발이 바닥에 닿는 자세
- **foot_l/r**: X+20 → 발바닥이 자연스럽게 바닥을 향하도록

### 척추 / 상체 기울기

```json
"spine_01":  [15, 0, 0],
"spine_02":  [10, 0, 0],
"spine_03":  [5, 0, 0]
```

- 척추를 단계적으로 앞으로 기울여 상체가 자연스럽게 숙여지는 자세 구현
- 총 약 30도 전방 기울기

### 턱을 괴는 팔 (오른손 기준)

```json
"clavicle_r":  [0, 0, -10],
"upperarm_r":  [-20, 0, -30],
"lowerarm_r":  [80, 0, 0],
"hand_r":      [-20, 0, 0]
```

- **clavicle_r**: 쇄골을 약간 내려 팔꿈치가 지지면에 닿는 자세 준비
- **upperarm_r**: X-20, Z-30 → 위팔을 앞으로 내리고 몸 쪽으로 모음
- **lowerarm_r**: X+80 → 팔꿈치를 거의 직각으로 굽혀 전완이 위를 향함
- **hand_r**: X-20 → 손이 아래로 약간 꺾여 턱에 자연스럽게 접촉

### 반대쪽 팔 (왼팔 — 자연스럽게 내린 상태)

```json
"clavicle_l":  [0, 0, 5],
"upperarm_l":  [0, 0, 15],
"lowerarm_l":  [10, 0, 0],
"hand_l":      [0, 0, 0]
```

- 왼팔은 무릎 위나 옆에 자연스럽게 놓인 상태

### 목 / 머리

```json
"neck_01":  [-5, 0, 0],
"head":     [-10, 5, 0]
```

- **neck_01**: X-5 → 목을 약간 앞으로 기울임
- **head**: X-10, Y+5 → 머리가 살짝 앞으로 숙여지며 약간 측면을 바라보는 자세 (생각하는 표정과 어울림)

## Step 2: ControlNet 강도 추천

### 권장값: **0.75 ~ 0.85**

**이유:**

앉아서 턱을 괴는 포즈는 다음 특성을 가진다:

1. **복잡한 팔 굽힘 각도**: 전완이 직각으로 올라오고 손이 얼굴에 닿는 구조는 ControlNet이 정확히 반영해야 함 → 너무 낮으면 포즈가 풀려버림
2. **앉기 자세의 하체 구조**: 허벅지-종아리 접힘 각도가 중요하므로 어느 정도 강한 강도 필요
3. **자연스러운 표정/얼굴 영역**: 강도가 너무 높으면(0.9+) 얼굴 디테일이 뭉개지거나 손-턱 접촉 부위가 부자연스럽게 표현될 수 있음

| ControlNet 강도 | 효과 |
|-----------------|------|
| 0.60 ~ 0.70 | 포즈가 느슨하게 반영됨. 앉기 자세가 서있는 자세로 무너질 위험 |
| **0.75 ~ 0.85** | **권장**: 포즈 충실도와 캐릭터 자연스러움의 균형 최적 |
| 0.90 ~ 1.00 | 포즈는 정확하나 얼굴/손 디테일이 딱딱해 보일 수 있음 |

### 시작 권장 설정

```
ControlNet 강도: 0.80
KSampler 스텝: 28
CFG Scale: 7.0
이미지 크기: 768×1024 (세로형 — 앉은 전신샷에 적합)
```

## 추가 팁

- **의자 표현**: 의자 자체는 ControlNet 스켈레톤에 포함되지 않으므로 프롬프트에 `sitting on chair, chair visible` 등을 명시해야 함
- **손 표현 주의**: 턱을 괴는 손 부분은 DWPose가 정확히 잡기 어려울 수 있음. 결과물에서 손이 어색하면 `hand_r` 본을 ±5~10도 조정하며 반복 테스트 권장
- **프롬프트 보조**: `sitting, chin resting on hand, elbow on knee, thoughtful pose` 등의 포즈 설명 태그를 긍정 프롬프트에 추가하면 ControlNet과 시너지 효과
- **3/4 뷰 권장**: 정면보다 약간 틀어진 각도(head Y+5~10)가 턱 괴기 포즈를 더 자연스럽게 표현함
